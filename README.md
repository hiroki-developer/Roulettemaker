# RouletteMaker サポートページ

RouletteMaker をご利用いただきありがとうございます。

## アプリ概要
ルーレット形式で候補をランダムに選択できるシンプルなアプリです。
迷ったときの決定、順番決め、抽選などにご利用ください。

## 主な機能
- 候補を自由に追加・編集
- ルーレット回転アニメーション
- 自動停止機能
- 振動（バイブレーション）演出
- Pro版で候補数上限アップ・強化演出
